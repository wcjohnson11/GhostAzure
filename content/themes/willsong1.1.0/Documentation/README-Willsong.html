Willsong Ghost Theme

Thank you for grabing Willsong.

Installation

The installation of this theme is the same for every Ghost Theme out there. Go to the root of your "Ghost Installation > content > themes" and place a copy of the willsong folder in there. Restart Ghost and go to your Ghost settings by adding "/ghost" at the end of your url (The same place where you signed up). In general settings at the bottom select the theme and everything is ready to go.

Cover and Logo

The cover and logo can be selected in the "general settings" tab in the admin area.

Disqus Forums

To start using the disqus comments, go to the partials folder and open the disqus.hbs file with the text editor of your choice and place your Disqus shortname in "your_shortname_here" variable. Go to Disqus.com and create an account in case you don't have one.
